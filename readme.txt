This file contains a brief description of the project that helps in navigating through the application.

First, the user opens the client app, and enters the IP of the server. Then, the login screen appears where the user is prompted to enter his/her username and password. If it is the first time for the user to use the application then she/he has the option to register for a new account. In the registration the user is asked to enter his name, address, email, username and password. Otherwise, the user has an account, and he is directed to the options page. The options page allows the users to share file via Client-Server architecture. Moreover, it contains local files button to display user files.

The user can download any of his files on the server, upload files to the server. Upon download or upload the progress is displayed in the notification.
